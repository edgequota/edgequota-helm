EdgeQuota Helm Chart

Distributed rate-limiting reverse proxy for Kubernetes.

rate-limiting reverse-proxy redis grpc http3 helm

Install

# Add the repository
helm repo add edgequota https://edgequota.github.io/edgequota-helm
helm repo update

# Install
helm install edgequota edgequota/edgequota -f values-minimal.yaml

Prerequisites

Kubernetes 1.28+ · Helm 3.12+ or Helm 4.x · Redis (external)
